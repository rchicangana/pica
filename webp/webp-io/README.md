## Usage

**Maven dependency**

```xml
<dependency>
    <groupId>io.github.biezhi</groupId>
    <artifactId>webp-io</artifactId>
    <version>0.0.3</version>
</dependency>
```

**API**

```java
String src  = "a.webp";
String dest = "a.png";
WebpIO.toNormalImage(src, dest);
```
